# Arashrasoulzadeh Tap

## How do I install these formulae?

`brew install arashrasoulzadeh/tap/<formula>`

Or `brew tap arashrasoulzadeh/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "arashrasoulzadeh/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
